# 핵심 기능 정의 (FSD)

> **작성일**: 2026-07-20

## MoSCoW

### Must
- 세션 시작 전 이름 입력(로그인 아님, 로컬 저장용 표시 이름)
- 배경 설문(Background Survey) 선택 화면
- 난이도(Self Assessment 1~6) 선택 화면
- 세트 구조 기반 문제 출제 엔진(콤보셋 2~3세트, 롤플레이, 돌발, 어드밴스)
- 문제 음성 재생(최대 2회) — TTS 또는 사전 녹음 음성
- 답변 녹음(MediaRecorder) + 자동 진행, 스킵(10초 후 활성화) 규칙
- 자유 연습 모드(주제/유형 골라서 시간제한 없이 연습, Phase 1)
- 자유 연습 문항 수 제한(세션당 5 또는 10문제 중 선택, 최대 10문제) — 필터 조건에 맞는 문제 중 무작위로 뽑아 구성
- 세션 종료 직후 규칙기반(임시) 점수 화면 — STT/LLM 없이 답변 완료율·평균 답변 길이로 대략적인 구간(NL~AL)을 즉시 보여줌. "실제 발화 내용은 분석하지 않은 임시 추정치" 고지 필수
- 배경 소음(시험장 웅성거림) on/off 토글 — 설정 화면에서 기본값 지정 + 연습 중 실시간 토글, 브라우저에 선호값 저장
- 유튜브 배경 영상 팝업 on/off — 유튜브 공식 embed(iframe)로 화면 구석에 작은 팝업 재생(다운로드·재배포 아님, 저작권 문제 없음)
- 전체 문제 목록 화면 — 문제은행에 등록된 모든 문제를 주제별로 훑어볼 수 있음
- 회차별 결과 목록(로컬 저장) 조회

### Should
- 실전 시뮬레이션 모드(전체 40분 타이머, 세트 순서 고정, 중간 난이도 재조정 1회)
- 녹음 재생(다시 듣기)
- STT 스크립트화 + 정량 피드백(발화량, 말속도, 필러 비율, 시제/접속사 키워드 다양성) — 규칙기반 점수를 대체

### Could
- LLM 기반 4대 기준 정성 피드백 + 참고용 예상 등급 — 규칙기반 점수를 대체
- 취약 유형 통계/오답노트(로컬 데이터 기반 집계)
- 문제은행 관리 화면(직접 문제 추가/편집)

### Won't (현 범위 밖)
- 회원가입/로그인, 서버 측 유저 계정
- 서버 DB에 유저 데이터 영속 저장
- 결제/구독
- 다국어 인터페이스(문제 자체는 영어, UI는 한국어 고정)

## 구조도 (Sitemap)

```
/                      홈 (모드 선택: 자유연습 / 실전시뮬레이션 / 기록보관함)
/practice/setup        자유연습 — 주제·유형 선택
/practice/session      자유연습 — 문제 진행
/settings/voice        TTS 보이스 미리듣기·선택·저장
/questions             전체 문제 목록(주제별)
/exam/survey           실전시뮬레이션 — 배경설문
/exam/level            실전시뮬레이션 — 난이도 선택
/exam/session          실전시뮬레이션 — 세트별 문제 진행(타이머 포함)
/exam/result/:id       회차 결과 요약(정량+LLM 피드백)
/history               기록보관함 — 회차 리스트
/history/:id           회차 상세(녹음 재생 + 피드백)
```

## Screen Spec

| 화면 | 구성 요소 | 설명 |
|---|---|---|
| 홈 | 모드 카드 3개, 최근 기록 요약 | 자유연습/실전시뮬레이션/기록보관함 진입점 |
| 배경설문(`/exam/survey`) | 카테고리별 체크박스(직업/여가/취미/운동/여행 등) | 선택 항목이 세트1~2 출제 풀이 됨을 안내 |
| 난이도 선택(`/exam/level`) | 1~6 단계 카드 + 설명 | 선택에 따라 12/15문항 안내 |
| 자유연습 설정(`/practice/setup`) | 이름 입력란, 문항 수 선택(5/10), 배경 소음 on/off, 주제·유형 체크박스 필터 | 이름 없으면 시작 버튼 비활성화 |
| 문제 진행(`/exam/session`, `/practice/session`) | 문제 텍스트/오디오 재생 버튼(재생 횟수 표시), 녹음 버튼, 타이머, 진행 인디케이터(N/전체), Skip 버튼(10초 후 활성), 배경 소음 실시간 on/off 토글 | 세트 경계에서 유형(콤보/롤플레이/돌발/어드밴스) 표시 |
| 보이스 설정(`/settings/voice`) | 사용 가능한 영어 보이스 목록 + 미리듣기 + 저장/자동선택 | 문제 듣기용 TTS 보이스를 직접 골라 저장 |
| 자유연습 즉시 결과(`/practice/session` 종료 시) | 추정 구간 배지, 답변 완료 수·완료율·평균 답변 길이 카드, 임시 추정치 고지 | STT/LLM 연동 전까지 규칙기반 점수만 표시 |
| 결과 요약(`/exam/result/:id`) | 상단 헤더(목표등급 vs 예상등급 배지 + 등급 사이 위치 progress bar), "목표등급 받기 위한 Checklist"(말하기속도/호흡당 문장길이/발화량 각각 Worst·Best·목표평균 막대비교+팁, 주제 적합도 경고, 문법 원문→교정문 대비+오디오 재생위치), "가장 못한 유형 및 주제"(다회차 집계), ACTFL 4대 기준 탭 + 보조 4축 레벨바(Complexity/Accuracy/Fluency/Coherence, 발음 제외), 재연습 CTA | "공식 결과 아님, AI 추정치" 고지 상시 노출 |
| 기록보관함(`/history`) | 회차 리스트(날짜/모드/난이도/예상등급), 필터 | 로컬 저장 데이터 기반 |
